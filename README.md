## Description
This is a Graphql API for resturant management system called (rms) for sahel resturant in my university ([chamran-university](https://scu.ac.ir/))


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
